: Windows

: Linux

- Requisitos
: Java 1.7
: Maven 3.x
: Git 2.x
: Banco de Dados (MySQL, Postgres, Derby entre outros)

- Instalarção/Run
: git clone git@github.com:tassiovirginio/reservarecursos.git
: cd reservarecursos
: mvn jetty:run
